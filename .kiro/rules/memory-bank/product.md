# Product Overview

## Project Purpose
STACK is a Web3-native investment platform designed for Gen Z users, bridging traditional finance and Web3 through a hybrid model that enables instant wealth-building. The platform provides fiat-to-stablecoin on-ramps, seamless stock/ETF investments, and AI-powered financial guidance.

## Value Proposition
- **Safe & Frictionless**: Demystifies Web3 while outperforming legacy banking in speed and fairness
- **Gen Z Native**: Fast, social, intuitive experience matching digital-native expectations
- **Protective Guidance**: AI CFO provides automated insights and portfolio analysis
- **Hybrid Finance**: Combines accessibility of traditional finance with Web3 innovation

## Key Features

### User Onboarding & Managed Wallets
- Simple sign-up with automatic secure wallet creation
- No seed phrase complexity - custody abstracted away
- KYC/AML orchestration for compliance
- Passcode-based authentication

### Multi-Chain Stablecoin Deposits
- Support for EVM chains (Ethereum, Polygon, BSC)
- Support for non-EVM chains (Solana)
- Automatic conversion to stablecoins for buying power
- Multi-chain wallet management

### Investment Flow
- Automatic stablecoin-to-fiat conversion
- Curated investment baskets (Tech Growth, Sustainability, ETFs)
- Simple portfolio view with performance tracking
- Integration with Alpaca for stock/ETF trading

### Virtual Accounts & Off-Ramping
- Due Network integration for virtual USD accounts
- USDC to USD off-ramp capabilities
- Seamless fiat withdrawals
- Real-time transaction tracking

### AI CFO (0G Integration)
- Automated weekly performance summaries
- On-demand portfolio analysis
- Risk assessment and diversification insights
- Uses 0G for inference and secure storage

### Brokerage Integration
- Alpaca integration for trade execution
- Secure custody of traditional assets
- Real-time market data
- Automated funding and settlement

## Use Cases

### Primary Use Cases
1. **Quick Onboarding**: Sign up, verify identity, create wallet in minutes
2. **Deposit Funds**: Transfer stablecoins from any supported chain
3. **Invest in Baskets**: Choose curated investment portfolios
4. **Track Performance**: Monitor portfolio with AI-powered insights
5. **Withdraw Funds**: Off-ramp to USD via virtual accounts

### Advanced Use Cases
1. **Multi-Chain Management**: Manage wallets across multiple blockchains
2. **AI-Guided Investing**: Receive personalized investment recommendations
3. **Automated Rebalancing**: AI CFO suggests portfolio adjustments
4. **Social Features**: Follow top traders (future phase)
